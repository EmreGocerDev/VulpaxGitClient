Token nasıl alınır?

GitHub Personal Access Token Alma

  1. GitHub.com'a giriş yapın
  2. Settings → Developer settings → Personal access tokens → Tokens (classic) veya Fine-grained tokens
  3. "Generate new token" butonuna tıklayın
  4. Token'a bir isim verin (ör: "Vulpax Git Client")
  5. Gerekli scope'ları seçin: repo, user, gist, notifications, admin:org, delete_repo, workflow
  6. "Generate token" butonuna tıklayın
  7. Token'ı kopyalayıp yukarıdaki alana yapıştırın
